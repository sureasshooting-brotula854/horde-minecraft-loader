## 📋 System Requirements

Your computer needs to meet these basic standards to run the server software smoothly:

- Windows 10 or Windows 11 operating system.
- At least 4 gigabytes of memory (RAM), though 8 gigabytes is better for performance.
- An internet connection to download the server files initially.
- The latest version of Java installed on your computer.

If you encounter issues during startup, verify that you have the latest Java Runtime Environment installed from the official website.

## 🛠 Using the Loader

The application window provides a clear interface for managing your server.

- **Check Updates:** The loader looks for new versions of the server software every time you open it. If a newer build exists, the software asks if you want to update. Choose "Yes" to keep your server running the latest features.
- **Start Server:** Once the loader confirms the server files are present, the "Start" button becomes active. Click this button to initialize the server process.
- **View Logs:** A window shows text output describing the status of your server. This helps you confirm that the server loaded correctly or identify why a crash occurred.
- **Stop Server:** Always use the "Stop" button in the application before you close the window. This saves your world state and prevents file corruption.

## 📁 File Locations

The application creates a dedicated folder in your Documents directory. You can find your world save files, configuration settings, and server logs within this folder structure. If you need to back up your world, copy these files to a separate location, such as a cloud storage service or an external hard drive.

## ❓ Frequently Asked Questions

**Does this software cost money?**
No, this tool is free and open for public use.

**How do I make my server public?**
Sharing a server with people outside your home network requires configuring your internet router. Look for instructions on port forwarding for your specific router model if you want friends to join from other locations.

**What happens if the loader does not start?**
Check your antivirus logs first. Sometimes security software flags new applications as potential threats. If the loader fails to open, ensure you run it with administrator permissions by right-clicking the icon and selecting "Run as administrator."

**Can I run multiple servers at once?**
This tool manages one server instance at a time to prevent conflicts with your computer's memory. To run multiple servers, change the installation directory for each instance and ensure you use different network ports for each one.

**Is this safe to use?**
The loader only interacts with the official server files. It does not modify your system files or change your Windows registry settings.

## 🔧 Troubleshooting

If you see an error message, read the log window inside the application. Many issues stem from missing system memory or an outdated Java version. Ensure that no other programs are currently using the network port intended for your Minecraft server. If the issue persists, delete the local folder and run the loader again to perform a clean download of the server files.
